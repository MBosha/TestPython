7.1 Переключение режима управления.

Для случаев, когда работа Системы RAZ в автоматическом режиме по каким-то причинам не может быть осуществлена, во всех Узлах предусмотрен ручной режим управления.

Переключатель режима управления «Автомат/Ручной» MAN-SA1 расположен внутри шкафа управления.



Сообщение о включении ручного режима управления отображается на панели в области системных сообщений:




ВНИМАНИЕ!
Панель управления в данном случае перестает выполнять функции управления и может использоваться только для контроля параметров.


ВНИМАНИЕ!
Если панель управления находится в выключенном состоянии, контроль параметров Системы должен производиться по показаниям датчиков, установленных по месту.


ВНИМАНИЕ!
Переключение режима управления должно осуществляться только когда Система полностью остановлена.
В противном случае переключение режима повлечет за собой отключение всего работающего оборудования.
