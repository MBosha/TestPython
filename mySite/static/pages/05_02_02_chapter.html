5.2.2 Запрет запуска Системы без сетей.

По умолчанию система управления запрещает произвести запуск, если не запущен хотя бы один сетевой контур или контур ГВС. Чтобы это требование убрать, необходимо изменить параметр:
Страница РАЦИОНАЛ Уставки
Группа 2. Система RAZ
Подгруппа 9. Настройки. Раздел №3
Параметр 205. Запрещать запуск Системы без работающих сетей [1 — Да, 2 — Нет]

1 — Да Запуск Системы RAZ будет запрещен если не запущена ни один сетевой контур или контур ГВС.
2 — Нет Запуск Системы RAZ возможен без запуска сетевых контуров и контуров ГВС.

ВНИМАНИЕ!
Устанавливать значение параметра "205" в значение "2 - Нет" рекомендуется только только во время проведения пуско-наладочных работ.
ВНИМАНИЕ!
Если значение параметра "205" имеет значение "2 - Нет" при работе котлов может отсуствовать проток в котловом контуре
