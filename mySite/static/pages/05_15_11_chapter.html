5.15.11 Баки-аккумуляторы в контуре ГВС.

Узел R-4BA обеспечивает подачу горячей воды потребителю и циркуляцию греющей через баки-аккумуляторы. Узел включает в себя две пары насосов - подогрева и подачи. Для обеих пар насосов организовано переключение по времени наработки и функция АВР по выходу из строя рабочего насоса.
Настройки производятся на странице:
Страница РАЦИОНАЛ Уставки
Группа 6. Узлы R-4
Подгруппа 58. R-4.BA Настройки
Параметр 1426....1447

Узел R-4BA запускается совместно с Узлом R-4.
В баках-аккумуляторах устанавливаются датчики уровня (аналоговые или дискретные) и датчики температуры.

ВНИМАНИЕ!
Тип датчика уровня в баке определяется при заказе Системы RAZ
Датчики подключаются к шкафу R-1 Online.

При использовании аналоговых датчиков уровень воды в каждом баке контролируется по всей высоте бака. Выходной сигнал датчика – 4…20 мА.
Страница РАЦИОНАЛ Уставки
Группа 6. Узлы R-4
Подгруппа 58. R-4.BA Настройки
Параметр 1426. Баки-аккумулятор. Подпитка.Включение. Уровень воды меньше, %
1427. Баки-аккумулятор. Подпитка.Выключение. Уровень воды больше, %
1428. Контур подачи. Выключение. Бак аккумулятор. Уровень воды меньше, %

1426 - Уровень в баке, при котором включается подпитка и открывается клапан подпитки баков из водопровода (в Узле R-3)
1427 - Уровень в баке, при котором отключается подпитка и закрывается клапан подпитки баков из водопровода (в Узле R-3)
1428. Уровень, при котором выключаются насосы подачи – минимальный аварийный уровень. По этому уровню происходит останов насосов подачи воды из баков потребителю, остальное оборудование остается в работе. Выдается аварийное сообщение о минимальном аварийном уровне. Включение насосов подачи происходит при восстановлении уровня в баках до минимального уровня, установленного в параметре 1426.
1430. Уровень, при котором происходит останов греющих насосов – максимальный аварийный уровень. По этому уровню происходит останов греющих насосов (клапан подачи в Узле R-3 закрыт ранее). Выдается сообщение о максимальном аварийном уровне. Включение греющих насосов происходит при падении уровня до максимального уровня, установленного в параметре 1427.

При использовании дискретных датчиков уровня, на каждый бак устанавливается комплект чувствительных элементов формирующих выходной сигнал типа «сухой контакт» и обеспечивающих контроль четырех уровней:
Максимальный аварийный уровень
Уровень, при котором происходит останов греющих насосов (клапан подачи в Узле R-3 закрыт ранее). Выдается сообщение о максимальном аварийном уровне. Включение греющих насосов происходит при падении уровня до появления сигнала максимального уровня.
Максимальный уровень
Уровень, при котором закрывается клапан подпитки баков из водопровода (в Узле R-3).
Минимальный уровень
Уровень, при котором открывается клапан подпитки баков из водопровода (в Узле R-3).
Минимальный аварийный уровень
Уровень, при котором происходит останов насосов подачи воды из баков потребителю, остальное оборудование остается в работе. Выдается аварийное сообщение о минимальном аварийном уровне.
Включение насосов подачи происходит при восстановлении уровня в баках до минимального уровня.
