5.7.6 Настройки регулятора.

Чтобы определить скорость реакции регулятора на изменение температуры, выберете из выпадающего списка регулирование:
― Нормальная нагрузка
― Низкая нагрузка
― Определять нагрузку автоматически
― Задать коэффициенты
Варианты работы регуляторов
Параметр Описание

Нормальная нагрузка.
Используйте этот режим, если хотите, чтобы регулятор быстро реагировал на изменение температуры на выходе из котла. Используйте ползунок «Быстрее/Медленнее» под выпадающим списком, чтобы подстроить скорость регулирования.

Низкая нагрузка.
Используйте этот режим, если хотите, чтобы регулятор отрабатывал плавно изменение температуры на выходе из котла. Используйте ползунок «Быстрее/Медленнее» под выпадающим списком, чтобы подстроить скорость регулирования.

Определять нагрузку автоматически.
Используйте этот режим для автоматической смены режима «Нормальная нагрузка» и «Низкая нагрузка».
Характер нагрузки определяется по времени переходного процесса.

Настройте работу регулятора на нормальной и малой нагрузках перед тем как выбрать этот режим работы.

Задать коэффициенты.
Используйте этот режим, чтобы вручную определить скорость работы регулятора, задав свои коэффициенты ПИД-регулятора.

Список параметров регулятора температуры котла. Модулируемое управление
Параметр Описание
Kp, Ti, Td Коэффициенты ПИД-регулятора
Выбег ИМ Время выбега сервопривода горелки. Задается в параметре:
Горелка. Время набора мощности от 0 до 100% при модуляции
Ход ИМ мин Определяет нижнее ограничение мощности горелки.
Параметр не может быть меньше значения параметра:
Горелка. Стартовая мощность при модуляции
Ход ИМ макс Определяет верхнее ограничение мощности горелки.
Параметр не может быть больше 100%
Зона нечувств. Значение зоны нечувствительности регулирования.

ВНИМАНИЕ!
Значения настроечных параметров регулятора в области ответственности специалиста по пуско-наладке.
