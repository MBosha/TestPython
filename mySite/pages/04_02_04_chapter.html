4.2.4 Вход пользователя «Газовщик»

Вход пользователя под учетной записью «Оператор» осуществляется по паролю через диалог «Доступ» главного меню:

ВНИМАНИЕ!
По умолчанию в Системе заведена учетная запись Газовщик
Пароль: 123789
ВНИМАНИЕ!
При утере пароля учетной записи Газовщик восстановление пароля возможно только с помощью специалистов РАЦИОНАЛ.

Установленный по умолчанию, на заводе РАЦИОНАЛ, пароль 123789. Можно изменить со страницы «Учет. Газ. Уставки» пользователю с правами доступа «Газовщик», который, как правило, является представителем газовых служб.

ВНИМАНИЕ!
За более подробной инструкцией обратитесь к документу
«UTG12. Руководство по монтажу и эксплуатации».
