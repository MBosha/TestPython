4.2.6 Доступ «Инженер» по умолчанию

Для того чтобы панель управления загружалась с уровнем доступа «Инженер» по умолчанию, например, во время проведения ПНР, надо изменить значение параметра:
Страница РАЦИОНАЛ Уставки
Группа 2. Система RAZ
Подгруппа 7. Настройки. Раздел №1
Параметр 165. Уровень доступа «Инженер» по умолчанию [1-Да, 2-Нет]

1-Да При перезагрузке панель будет загружаться под уровнем доступа "Инженер"
2-Нет При перезагрузке панель будет загружаться под уровнем доступа "Оператор"
