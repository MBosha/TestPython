4.4.4 Страница «Рационал. Настройка каналов. AI». Настройка входных аналоговых сигналов

Входным аналоговым сигналом (AI — Analog Input) в системе ввода-вывода считается сигнал с датчика, измеряющего тот или иной технологический параметр. Стандартный сигнал с датчика (токовый 4..20 мA, сигнал с термосопротивления) поступает на канал модуля ввода-вывода, где оцифровывается и передается в систему. Из полученного оцифрованного значения системой управления вычисляется действительное значение технологического параметра.
На странице «Рационал. Настройка каналов. AI» предоставляется возможность произвести диагностику и настройку параметров масштабирования сигнала. В общем виде страница состоит из двух информационных областей:
– Список входных аналоговых сигналов. Содержит полный список сигналов, определенных в Системе в зависимости от текущей комплектации котельной;
– Параметры сигнала.
Доступ к сигналам AO осуществляется со страницы «Рационал. Настройка каналов. AI»



При перемещении по списку аналоговых сигналов, определенных в системе, инженеру предоставляется возможность изменения параметров выбранного элемента. Список параметров приведен в таблице ниже.

Параметры входного аналогового сигнала
Параметр Описание
Индекс Номер сигнала в общем массиве сигналов типа AI.
Используется для числовой идентификации сигнала.
Значение Вычисленное значение сигнала
Состояние сигнала Состояние сигнала Норма/Ошибка
Состояние модуля Флаг диагностики модуля ввода-вывода, на который поступает сигнал/td>
Состояние канала Флаг диагностики канала ввода-вывода, на который поступает сигнал
Шкала минимум Нижний предел шкалы датчика. Данный параметр активен только для датчиков с токовым сигналом
Шкала максимум Верхний предел шкалы датчика. Данный параметр активен только для датчиков с токовым сигналом
Код минимум Минимальное значение оцифрованного сигнала в кодах
(код «0» соответствует значению сигнала 0mA, код «6500» соответствует значению сигнала 4 mA). Данный параметр активен только для датчиков с токовыми сигналами
Код максимум Максимальное значение оцифрованного сигнала в кодах.
(код «32665» соответствует значению сигнала 20mA). Данный параметр активен только для датчиков с токовыми сигналами
Входное значение Отображает в кодах оцифрованное значение поступающего на вход аналогового сигнала («сырое» значение)
Признак маскирования Признак принудительной записи пользовательского значения сигнала. В случае установки флага, сигнал принимает значение, установленное в поле «Значение маскирования»
Значение маскирования Пользовательское значение, принудительно назначаемое сигналу/td>
Коэффициент масштабирования Коэффициент масштабирования активен только для температурных датчиков. Вычисление значения температурного сигнала осуществляется простым умножением «сырого» значения сигнала на данный коэффициент
Постоянная времени фильтра, сек В обработке входного сигнала используется экспоненциальный фильтр, сглаживающий резкие скачки значения измеряемого параметра. Постоянная времени фильтра определяет «пологость» сглаживания измерения
Тип модуля Тип модуля ввода-вывода, к которому привязывается сигнал.
Модуль может быть:
– AI. Модуль токовых сигналов 4..20mA;
– AT. Модуль температурных сигналов с термосопротивлений Pt100;
- MIX. Смешанный модуль с AI-каналом 4..20mA
Номер модуля Номер модуля аналогового ввода в станции, к которому привязывается сигнал
Номер канала Номер канала в модуле аналогового ввода, к которому привязывается сигнал

ВНИМАНИЕ!
Перед первым запуском необходимо проверить привязки сигналов (Тип модуля, Номер модуля, Номер канала) - на соответствие проекту и монтажу в шкафу управления R-1 Online.
